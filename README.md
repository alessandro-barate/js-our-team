Ogni membro è caratterizzato dalle seguenti informazioni: nome, ruolo e foto.

MILESTONE 2:
Stampare le stesse informazioni su DOM sottoforma di stringhe.

BONUS 1:
Trasformare la stringa foto in una immagine effettiva.

BONUS 2:
